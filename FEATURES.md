# ✨ Особливості лендінг-сторінки Schedulify

## 🎨 Дизайн

### Кольорова схема
- **Основний колір**: `#7092bf` (синій з demo-org.json)
- **Темний відтінок**: `#5a7a9f`
- **Світлий відтінок**: `#8faad1`
- Повністю відповідає брендингу demo-org

### Іконки
- **Material Icons** від Google
- Понад 2000+ доступних іконок
- Чіткі SVG іконки для всіх роздільних здатностей
- Використані іконки: calendar_month, event_busy, notifications_active, security, phone_iphone, groups, і багато інших

### Типографіка
- **Шрифт**: Roboto (Google Fonts)
- **Ваги**: 300 (Light), 400 (Regular), 500 (Medium), 700 (Bold)
- Оптимізовано для читабельності на всіх пристроях

## 📱 Адаптивний дизайн

### Брейкпоінти
- **Десктоп**: 1200px+
- **Планшет**: 768px - 1199px
- **Мобільний**: < 768px
- **Малий мобільний**: < 480px

### Мобільна навігація
- Бургер-меню для екранів < 768px
- Плавна анімація відкриття/закриття
- Автоматичне закриття при кліку на посилання
- Закриття при кліку поза меню

### Адаптивні сітки
- Автоматична перебудова для різних екранів
- 3 колонки (десктоп) → 2 колонки (планшет) → 1 колонка (мобільний)
- Оптимізований відступ між елементами

## 🎨 CSS Анімації

### Hover ефекти
```css
- Картки: translateY(-8px) + збільшена тінь
- Кнопки: translateY(-2px) + тінь
- Навігація: підкреслення знизу
- Контактні картки: підняття + тінь
```

### Keyframe анімації
```css
- Float анімація для зображення в hero
- Плавні переходи (0.3s ease)
- Transform та opacity анімації
```

### Responsive breakpoints
```css
- 768px: Мобільне меню, одноколонкові сітки
- 480px: Оптимізація для малих екранів
```
